# HireReady – Landing Page (Challenge S12)

Este repositorio contiene el avance de la Landing Page del proyecto **HireReady**, una plataforma que utiliza inteligencia artificial para ayudar a estudiantes y egresados a prepararse para entrevistas laborales mediante simulaciones, análisis de voz, lenguaje corporal y retroalimentación en tiempo real.

Este desarrollo corresponde al **Challenge de la semana 12** del curso *IHC y Tecnologías Móviles*.

## Objetivo del proyecto

El objetivo de este challenge es implementar una versión inicial de la Landing Page de HireReady que permita:

- Comunicar claramente **qué es HireReady**.
- Presentar el **problema** que resuelve: falta de preparación para entrevistas.
- Mostrar **beneficios** y funcionalidades clave.
- Seguir la estructura de carpetas solicitada por el curso.
- Evidenciar el uso de **Git**, **GitHub**, y el flujo de trabajo mediante ramas (`main` y `develop`).

---

## Funcionalidades implementadas en este avance

- Sección **Hero** con descripción del producto y CTA (“Probar ahora”).
- Sección **Testimonios** con tarjetas e imágenes reales (formato PNG).
- Estilos organizados en `styles.css` siguiendo buenas prácticas.
- Estructura semántica en HTML5.
- Archivos organizados según lo solicitado por el profesor.
- Implementación del control de versiones:
    - Rama `main` → versión estable.
    - Rama `develop` → rama activa para avances.
